# Oğuz Ölke - HW 2

## Hangman Game

### Rules

- User has 5 guesses
- User can only enter one "alphabetic" letter at a time

In order to run this project

```bash
git clone https://github.com/patika-hepsiburada-react-bootcamp/odev-2-olkeoguz.git
```
And then
```bash
npm install
```
to install all the dependencies
